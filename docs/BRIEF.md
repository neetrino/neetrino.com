# Техническое задание — Neetrino

## Описание

Публичный сайт IT-компании Neetrino: главная, услуги, портфолио, о компании, контакты. Верстка с опорой на макет Figma, акцент на визуал и производительность (Next.js App Router).

## Целевая аудитория

Клиенты и партнёры, ищущие разработку сайтов, мобильных приложений, CRM и интеграций с AI.

## Основные функции (приоритет)

1. Статический контент и навигация по разделам — высокий
2. Формы обратной связи (UI; бэкенд по мере подключения) — средний
3. Анимации и 3D-блоки на главной — средний

## Стек (зафиксировано)

См. `docs/02-TECH_STACK.md` и `docs/TECH_CARD.md`.

## Дизайн

- Источник: Figma (экспорт ассетов в репозитории).
- Стили: Tailwind CSS 4, shadcn/ui, кастомные шрифты.

## Интеграции

- По мере необходимости: хостинг (например Vercel), аналитика — уточняются в TECH_CARD.

## Язык интерфейса

- Основной: английский (контент в компонентах).

## Ограничения

- Без обязательного backend в текущей фазе лендинга.
- Секреты только через переменные окружения, не в коде.
