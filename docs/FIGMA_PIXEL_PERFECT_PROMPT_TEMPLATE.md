# Pixel-Perfect Figma Prompt Template

Чтобы агент сразу сделал страницу максимально близко к Figma, в промпте нужно жестко зафиксировать:

- взять полный `get_design_context` код без упрощений;
- сохранить absolute layout, все сырые px-значения и `data-node-id`;
- рендерить как десктоп-канвас `1440x2174` через scale-wrapper;
- выполнить обязательное сравнение скриншотом и править до совпадения.

## Готовый шаблон

```md
Сделай страницу PORTFOLIO pixel-perfect по Figma URL: https://www.figma.com/design/peTGKUcaTLyATMJ5xJVaXl/NEETRINO-WEB?node-id=166-1203&m=dev.

Правила:

1. Обязательно вызови get_design_context(fileKey,nodeId) и используй код из ответа как единственный источник правды.
2. НЕ упрощай и НЕ перерабатывай верстку: сохрани absolute layout, все точные px-значения, порядок/вложенность, data-node-id.
3. НЕ делай responsive-адаптацию этой страницы. Сделай канвас ровно 1440x2174 и масштабируй его через scale-wrapper (CanvasScaler).
4. Допустимы только минимальные адаптации:
   - img -> Image unoptimized
   - a -> Link только если это не ломает позиционирование
   - добавь 'use client' при необходимости
5. Ассеты оставь константами в этом же файле.
6. После реализации:
   - открой страницу в браузере,
   - сделай скрин,
   - сравни с screenshot из get_design_context,
   - исправляй до визуального совпадения.
7. В конце дай отчет: что совпало/что поправил + путь к файлу.
```
