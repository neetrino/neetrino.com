# Archived Neetrino home reference

`NeetrinoHome.full.tsx` is **not** part of the Next.js production bundle. It is kept for Figma export scripts under `scripts/neetrino/` (`extract-neetrino-segments.mjs`, `build-neetrino-below-fold.mjs`, etc.).

When adding assets, prefer updating `lib/figma-assets.ts` for runtime code.
